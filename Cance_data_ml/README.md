# Cancer Prediction using Machine Learning

## Overview
This project predicts cancer diagnosis using a K-Nearest Neighbors (KNN) classifier.
It demonstrates a complete data science workflow: EDA, preprocessing, outlier handling,
model training, and evaluation.

## Dataset
- Breast Cancer Wisconsin Dataset
- Target variable: diagnosis (Benign / Malignant)

## How to Run
```bash
pip install -r requirements.txt
python src/train.py
python src/evaluate.py
```
